# MineWorld 3D

这是一个完全基于 HTML5、Three.js 和 Web Audio API 构建的 3D《我的世界》网页版克隆项目。
包含纯代码合成的正宗物理音效、程序化地形生成（无限挖掘）、以及真正的跨国 WebSocket 多人联机功能！

## 功能特点
- **完全方块化材质**：没有绝对的圆形，所有贴图（草地、原木、泥土）均为 16x16 像素噪点随机生成。
- **极致的无限挖掘**：采用动态表面剔除算法，向下挖 10000 层也不会卡顿。
- **纯粹的合成音效**：没有使用任何现成的 MP3 录音。泥土的“沙沙”声、石头的“咣咣”声、玻璃破碎的“叮当”声全部通过 Web Audio API 纯代码实时物理合成。
- **多人联机**：自带一个轻量级的 WebSocket 服务器，支持玩家输入公网 IP 进行真实的跨国联机。可以看到彼此的动作并同步破坏/放置方块。
- **丰富的机制**：包含核弹爆炸、枪械射线检测、火把供电的熔炉、村民建造彩蛋、以及水下窒息（20秒）等物理机制。

## 如何运行单人游戏
由于使用了本地文件系统和 `fetch`（或者本地资源），你不能直接双击打开 HTML 文件，需要启动一个本地服务器：

1. 确保你安装了 Python 3 或 Node.js
2. 在项目根目录下打开终端
3. 运行本地服务器：
   - Python: `python3 -m http.server 8000`
   - Node.js: `npx serve -p 8000`
4. 打开浏览器访问：[http://localhost:8000/game3d.html](http://localhost:8000/game3d.html)

## 如何开启多人联机服务器
如果你想和朋友联机，需要运行后端的 WebSocket 服务器。

1. 安装依赖 (需要 Node.js 或 Python 环境):
   如果你使用 Python (推荐):
   `pip install websockets`
   
2. 启动联机服务器:
   `python3 server.py`

3. 游戏内操作:
   - 在游戏中点击“联机游戏”。
   - 界面上会自动显示你的公网 IP 地址。
   - 把你的公网 IP 发给朋友。
   - 朋友在他们的游戏里点击“联机游戏”，输入你的 IP 并点击连接。

## 如何部署到云端让所有人都能玩
如果你想让全世界的人都能玩你的游戏，你需要将前后端分离部署：

1. **前端部署 (HTML)**:
   - 将此仓库推送到 GitHub。
   - 在 GitHub 仓库设置中开启 **GitHub Pages**。
   - 别人就可以直接通过 `https://你的用户名.github.io/仓库名/game3d.html` 来玩了。

2. **后端部署 (server.py)**:
   - 由于 GitHub Pages 不支持运行 WebSocket 服务器，你需要将 `server.py` 部署到如 **Render**, **Railway**, 或 **Heroku** 等免费云服务器平台上。
   - 部署后会获得一个类似于 `wss://your-app.onrender.com` 的地址。
   - 在游戏中连接这个地址即可联机！
